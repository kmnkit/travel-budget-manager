# Travel Expense Tracker（旅行支出トラッカー）

海外旅行者が現地での支出を簡単に記録・管理できるクロスプラットフォームアプリケーション。

## 🌟 主な機能

### Webアプリ（完成）
- ✅ **認証・アカウント管理**: ログイン、サインアップ、プロフィール管理
- ✅ **オンボーディング**: 5ステップの初回チュートリアル
- ✅ **旅行管理**: 旅行の作成、編集、削除、詳細表示
- ✅ **支出記録**: カテゴリー別支出管理、詳細・編集機能
- ✅ **カテゴリー管理**: カスタムカテゴリー作成（絵文字・カラー選択）
- ✅ **レポート・分析**: Rechartsによるグラフ表示、統計情報
- ✅ **検索・フィルター**: 旅行・支出の検索、カテゴリー・金額・日付フィルター
- ✅ **予算管理**: 予算設定、使用率表示、残高トラッキング

### モバイルアプリ（開発予定）
- 💳 **ICカード読み取り**: Suica、Pasmo、Octopus等の交通系ICカード対応
- 📸 **レシートOCR**: カメラで撮影して自動データ抽出
- 📱 **プッシュ通知**: 予算超過アラート、旅行リマインダー
- 🌐 **オフライン対応**: ネットワークなしでもデータ記録可能
- 📍 **位置情報連携**: 支出場所の自動記録

## 🎨 デザイン

赤を基調とした情熱的で使いやすいUI/UXデザイン

## 🛠️ 技術スタック

- **iOS**: SwiftUI
- **Android**: Kotlin + Jetpack Compose
- **Web**: React + TypeScript + Tailwind CSS
- **バックエンド**: Supabase (認証、データベース、ストレージ)
- **OCR**: Vision Framework (iOS)、ML Kit (Android)、Tesseract.js (Web)
