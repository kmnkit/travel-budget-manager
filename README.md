# Travel Expense Tracker（旅行支出トラッカー）

海外旅行者が現地での支出を簡単に記録・管理できるクロスプラットフォームアプリケーション。

## 🌟 主な機能

- 📸 **レシート/領収書のOCR読み取り**: カメラで撮影するだけで自動的に支出を記録
- 🗂️ **旅行ごとの管理**: 各旅行を個別に管理し、予算をトラッキング
- 💰 **予算管理**: 旅行ごとに予算を設定し、リアルタイムで残高を確認
- 📊 **詳細なレポート**: カテゴリー別・日別の支出分析、グラフ表示
- 🏷️ **カテゴリー＆タグ**: 支出を分類して整理
- 💳 **支払い方法記録**: 現金、クレジットカードなど支払い方法を記録
- 📱 **マルチプラットフォーム**: iOS、Android、Webアプリで同じデータを共有

## 🎨 デザイン

赤を基調とした情熱的で使いやすいUI/UXデザイン

## 🛠️ 技術スタック

- **iOS**: SwiftUI
- **Android**: Kotlin + Jetpack Compose
- **Web**: React + TypeScript + Tailwind CSS
- **バックエンド**: Supabase (認証、データベース、ストレージ)
- **OCR**: Vision Framework (iOS)、ML Kit (Android)、Tesseract.js (Web)
