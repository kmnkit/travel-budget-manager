## 한국어

### 소개

Trip Wallet(이하 "앱")은 사용자의 개인정보 보호를 중요하게 생각합니다. 본 개인정보 처리방침은 앱 사용 시 사용자의 정보가 어떻게 처리되는지 설명합니다.

### 수집하는 정보

**Trip Wallet은 어떠한 개인정보도 외부 서버로 수집, 저장, 전송하지 않습니다.**

앱에 입력하는 모든 데이터:
- 여행 정보 (이름, 날짜, 예산)
- 지출 기록 (금액, 카테고리, 결제 수단)
- 환율 설정

는 SQLite 데이터베이스를 사용하여 **사용자 기기에만 로컬 저장**됩니다.

### 제3자 서비스

**환율 API**: 앱은 open.er-api.com에서 환율 정보를 가져옵니다. 이 서비스는 IP 주소 등 익명화된 사용 데이터를 수집할 수 있습니다. 자세한 내용은 해당 서비스의 개인정보 처리방침을 참조하세요.

### 데이터 보안

사용자의 데이터는 기기에 로컬로 저장되며, 기기의 내장 보안 기능으로 보호됩니다. 개발자는 사용자의 데이터에 접근할 수 없습니다.

### 데이터 보존

모든 데이터는 다음 상황까지 기기에 보존됩니다:
- 앱 삭제
- 기기 설정에서 앱 데이터 삭제
- 앱 내 데이터 관리 기능 사용

### 아동 개인정보

Trip Wallet은 13세 미만 아동의 정보를 고의로 수집하지 않습니다. 본 앱은 모든 연령에 적합합니다.

### 정책 변경

본 개인정보 처리방침은 수시로 업데이트될 수 있습니다. 변경 사항은 "최종 수정일"에 반영됩니다.
