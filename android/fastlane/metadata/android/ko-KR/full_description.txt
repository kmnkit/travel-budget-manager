My Trip Wallet은 여러 통화로 여행 경비를 쉽게 관리할 수 있도록 도와줍니다.

주요 기능:
• 다중 통화 지원 (원화, 달러, 유로, 엔화, 파운드, 호주달러, 캐나다달러)
• 실시간 환율 업데이트
• 시각적 알림을 통한 예산 추적
• 8가지 지출 카테고리 (식비, 교통비, 숙박비 등)
• 다양한 결제 수단 (현금, 카드, 모바일 결제)
• 아름다운 차트와 통계
• 이중 통화 표시 (원래 금액 + 환산 금액)
• 오프라인 우선: 모든 데이터가 로컬에 저장됨
• 광고 제거 프리미엄 옵션 제공

이런 분들에게 적합:
✓ 해외 여행자
✓ 출장자
✓ 예산을 신경 쓰는 여행자
✓ 다국가 여행자
✓ 지출 추적을 좋아하시는 분

개인정보 보호:
데이터는 기기에만 저장됩니다. 개인정보를 외부 서버로 수집, 저장, 전송하지 않습니다.

프리미엄 기능:
• 모든 광고 제거
• 개발 지원

지금 My Trip Wallet을 다운로드하고 여행 경비를 관리하세요!
